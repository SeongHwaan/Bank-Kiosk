## KGU BANK

이 프로그램은 은행업무 키오스크로써 사용자의 아이디와 비밀번호를 입력하면 현금입금, 현금인출, 계좌이체, 거래내역조회 등과 같은 은행업무를 수행할 수 있습니다.

## Installation
